# Sistema de auditoría de hardware 

Herramienta automatizada en Python diseñada para realizar auditorías técnicas de componentes de hardware (CPU, RAM, Disco) y diagnóstico del sistema. Es una solución eficiente compatible con sistemas operativos Windows y macOS.

##  Instalación y Ejecución

Para poner en marcha el sistema, sigue estos pasos desde tu terminal:

### 1. Requisitos
* Asegúrate de tener Python instalado.
* El proyecto requiere las dependencias listadas en el archivo `requisitos.txt`.

### 2. Configuración
Instala las librerías necesarias ejecutando el siguiente comando en la carpeta raíz:

```bash
pip install -r requisitos.txt

3. Ejecutar el sistema
Inicia el proceso de auditoría con el siguiente comando:

Bash
python main.py

Estructura del Proyecto
El repositorio está organizado de la siguiente manera:

src/: Carpeta que contiene el código fuente modular del sistema.
main.py: Script principal que inicia la auditoría.
requisitos.txt: Archivo con las dependencias necesarias para el funcionamiento.
.github/: Configuración de flujos de trabajo (workflows).
Desarrollado para facilitar el diagnóstico técnico de hardware de forma automatizada.
